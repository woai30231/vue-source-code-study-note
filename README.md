# 写本文档的目的
这里的源码基于vue 2.5.2版本，可在[这里](https://github.com/woai30231/vue)查看vue源码！本文档初次预定三个月利用工作之外业余时间来学习vue的源码，预计达到如下目的：1）熟悉vue源码结构；2）熟悉相关api的原理；3）学习作者的编程思路和思维（从代码中可窥探）。
